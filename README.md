# sentiment-analysis-webapp-frontend

> This is the frontend project of [sentiment-analysis-webapp](https://github.com/bighuang624/sentiment-analysis-webapp)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
